# Filled-project example

This folder is a **reference** showing what a real `docs/ai/` folder looks like
after the templates have been filled.

The example project is a small leave-management SaaS — chosen because it has:
- A real business domain (leaves, holidays, balances).
- A clear layer split (Domain / Application / Infrastructure / Interfaces).
- A mix of .NET backend + Vue frontend, so agents see both stacks.

Copy these files into your own `docs/ai/` as a starting point, then adapt to
your project. The shape and level of detail are what matter — not the
specific content.

```
examples/filled-project/
└── docs/
    └── ai/
        ├── PROJECT.md
        ├── ARCHITECTURE.md
        ├── COMMANDS.md
        ├── TESTING.md
        ├── DECISIONS.md
        ├── ROADMAP.md
        └── GLOSSARY.md
```
