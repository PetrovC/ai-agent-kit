Job name: router-parity
Runner: ubuntu-latest
Steps:
1. Checkout the repository (use actions/checkout at version 4)
2. Run an inline bash script that:
   a. Extracts all level-2 headings (lines starting with "## ") from each of the three router files

      any exception. If not, print it as a drift finding and set a failure flag.
   d. After scanning all headings, exit 1 if any drift was found, exit 0 otherwise.

checkout pattern used in this repo.
