### Background

Currently install.sh always installs everything (skills, hooks, settings, commands,
docs/ai/ templates, etc.). The `--profile minimal` flag will install only the
essential skills + router files, letting users adopt the kit gradually without
installing hooks, settings changes, or docs templates they don't want yet.

**Profile definitions:**
- `--profile full` (default): current behavior — unchanged
- `--profile minimal`: skills only + root router files (CLAUDE.md, AGENTS.md, AGY.md)
  — skips hooks, commands, settings.json, agents/, docs/ai/, audit, delegate
